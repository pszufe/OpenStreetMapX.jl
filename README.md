# OSMsim.jl
OpenStreetMap - routing and simulations library

## Installation

The current version uses Julia 0.7.0

Before using the library you need to install Julia packages, press `]` to go to package manager:

```julia
add Plots
add Distributions
add DataFrames
add DataFramesMeta
add FreqTables
add HTTP
add Query
add Shapefile
add LibExpat
add LightGraphs
```
